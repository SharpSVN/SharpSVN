SharpSvn is a free download and you can start using it today. If you do, contribute something back: answer somebody's question in the discussion forum or report a defect (how about fixing one? ). You can even become a core committer to the SharpSvn project.

Get access to the code and issue tracker

You can access the SharpSvn sourcecode from our repository https://ctf.open.collab.net/svn/repos/sharpsvn/trunk/ using the 'guest' account that has no password, but to enter and edit issues in our issue tracker, you need to request the "Participant" role in our project. To request the Participant role, you need to be an openCollabNet member and be logged in. We'll send you a confirmation email (within one working day, but typically much faster).
