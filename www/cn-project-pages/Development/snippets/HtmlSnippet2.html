To get access to the SharpSvn code and the ability to enter and edit issues in our issue tracker, you need to request the "Participant" role in our project. To request the Participant role, you have to be an openCollabNet member first, and be logged in. We'll send you a confirmation email (usually within one working day). For just checking out the source code you can use username 'guest' which has a blank password.
