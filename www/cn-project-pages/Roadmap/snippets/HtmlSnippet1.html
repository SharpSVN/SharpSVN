SharpSvn's original roadmap is completed. We implemented everything from our initial roadmap and SharpSvn is used by quite a few projects. For maintenance reasons we integrated the SharpSvn project with AnkhSVN (primary user and huge developer overlap). 

 

* Keeping SharpSvn stable.

* Implement new features of Subversion in a .Net friendly way.

* Support project using SharpSvn with some forward compatibility where possible by pro-actively adding features that will be standard in new versions.
